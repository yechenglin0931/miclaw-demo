21.02 · 一键执行类模板

  • 尺寸 2×1：最小的小部件，点击即触发完整 Agent 任务
  • 布局：左侧 icon（优先 MIUIX 图标库，兜底 AI 生成）+ 右侧双行文字（主标题 4-5 字 + 副标题描述）
  • 点击行为：直接在后台启动预设的 Agent 任务，无需用户额外输入
  • icon、标题、副标题由模板代码定义结构，内容由大模型基于上下文自动填充

21.03 · 自动化类模板

  • 尺寸 4×2 ~ 4×4：Agent 后台持续轮询，桌面呈现"数字仪表盘/监控雷达"
  • 内容：任务名称 + 任务概述 + 历史执行记录（高亮达标条件）+ Agent 状态 + 时间戳
  • 单次操作按钮：点击"立即执行一次"不影响原定的自动化频率
  • 达到预设阈值时自动触发系统原生通知

21.04 · 纯文本类模板

  • 尺寸 4×2 ~ 4×4：Agent 后台定时或事件触发，以 Markdown 流水账形式同步到桌面
  • 内容：主标题（4-15 字）+ 次要文本（最多 5 行，超出省略）+ 时间戳
  • 适用场景：每日工作结算板、周报自动生成、Git 提交记录汇总
